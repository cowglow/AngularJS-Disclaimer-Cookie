AngularJS Disclaimer Cookie

**AN EXAMPLE

This Service gets initiated from the controller DisclaimerCookie.initialize();

At page load the cookie will be undefined, conditional logic trigger display of the 'Disclaimer Template', DisclaimerCookie._template. Cookie is set when user clicks on 'X' button.

The value stored in the cookie is a timestamp string. The default value is 2 days ahead. This all depends on the user loading the page atleast one time a day.
